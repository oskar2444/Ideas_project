This repository contains the ZeroFOD Control project, developed as a university assignment to tackle the critical safety challenge of Foreign Object Debris (FOD). FOD refers to any misplaced particle, tool, or substance that can cause severe damage and safety hazards in sensitive environments. Our solution is a functional, easy-to-run prototype that utilizes AI and computer vision to provide real-time detection of these hazards. The system is designed to automatically classify objects such as metals, plastics, and tools, while instantly sending automated alerts when debris is detected. Additionally, the application features comprehensive event logging with timestamps and location markers, which powers trend analytics to help identify high-risk zones and recurring sources of debris.
